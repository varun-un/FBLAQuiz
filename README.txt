FBLA Quiz

Created by Varun Unnithan, from Morris Hills High School, for the FBLA 
Coding and Programming Competitive Event 2021.

All source codes and images, with the exceptions of those related to the 
third-party libraries listed below, are original works by Varun Unnithan.



USE
-------------------------
To run the program, open the FBLA Quiz.jar file within the program folder.
Follow the program's navigation to use the application.

Score reports that have been exported as PDFs can be found in the Score 
Reports folder within the project's main directory.

Program documentation can be found within the javadoc folder, which was 
created using Java's Javadoc documentation generator.
